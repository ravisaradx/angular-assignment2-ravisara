# angular-assignment2-ravisara

- attribute directive
  - เมื่อ hover ที่ข้อความจะมีข้อความใหม่ขึ้นมา
  - ลาก mouse เข้าออก พื้นหลังจะสลับสีระหว่างแดงและเขียว
  - ถ้ากดแป้นพิมพ์คำว่า 'rainbow' พื้นหลังจะเปลี่ยนสีเป็น แดง แสด เหลือง เขียว น้ำเงิน คราม ม่วง ตามลำดับ
- structural directive: มีช่องสำหรับตอบคำถาม หากตอบคำถามถูกต้อง จะมีข้อความปรากฏขึ้นมา
- pipe: เปลี่ยนคำต่าง ๆ เป็น emoji เช่น 'crab' เป็นรูปปู
